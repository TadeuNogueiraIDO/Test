What is it for?
This tool allows you to create a "Contact Us" form. This feature is very efficient, because it creates a communication channel between the user of your Ido and you.
In the form, the user can clear doubts, ask questions, send suggestions, in addition to being an excellent lead capture (user data), since he needs to fill in the fields that you activate.
How to use it?
1- Add a tool to your Ido and select "Contact Us".

2-Fill in the information on the screen:
  • Title: Enter a title for your form. Example: “Contact Us”, “Leave your question, comment or compliment” etc;
  • Description: Briefly explain what the user must do on the form. This field is optional;
2-Select the optional fields that will appear for the user to fill in:
  • Name;
  • Phone;
  • Subject;
The Email and Message selectors will already be active, as they are required for this tool.
