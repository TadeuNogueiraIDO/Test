What is it for?
This tool allows you to create a subscription form for sending Ido newsletters.
This feature is an excellent lead capture (user data) as it fills in and submits your information.
How to use it?
1- Add a tool to your Ido and select "Newsletter Form";

2- Fill in the information on the screen:
  • Title: Enter a title for your form. Example: “Subscribe to our newsletter”, “Get news”;
  • Description: Briefly explain what the user must do on the form. This field is optional;

3- Select the optional fields that will appear for the user to fill in:
  • Name;
  • Phone
The Email selector will already be active, as it is mandatory for this tool.
