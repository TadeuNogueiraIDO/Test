What is it for?
This tool allows you to add multiple images, forming a gallery of products, services, photographs, etc.
You can also configure one click for each image and when accessing, the user will be directed to WhatsApp, E-mail or external link.
It is still possible to create a detail page, with more information: title, description, price and a targeting button.
But stay tuned! With this feature it is not possible to make in-app purchases. To have this functionality and all the convenience of being an Ido store, purchase the Shop tool.
How to use it?
1- Add a tool to your Ido, select ”Image Carousel”;

2- Enter a title for your carousel and choose whether it will be visible to the user;

3- Select the image format:
  • Portrait: vertical rectangular images;
  • Landscape: horizontal rectangular images;
  • Square: same size vertically and horizontally (1x1);
It is important that the group of images have the same format so that there is no distortion when applied to the carousel.
4- Upload the images, selecting one at a time from your device.

5- Define if each image will have its details page. If yes, fill in:
  • Title;
  • Price;
  • Description;
  • Under Page button select one of the options:
    • No button: informational page only;
    • Open Whatsapp, then fill in the phone number;
    • Open email, then enter email;
    • Go to external link: Enter the URL of the website or social network

In case you don't want to create the detail pages, you can set the click action for each image:
  • No action: no click;
  • Open Whatsapp, then fill in the phone number;
  • Open email, then enter email;
  • Go to external link: Enter the URL of the website or social network
