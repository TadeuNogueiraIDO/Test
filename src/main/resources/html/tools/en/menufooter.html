What is it for?
The MENU or menu is where you focus what is most important to present to the user. It is in the MENU that we can add quick access to Tools, Product Categories and external links.
The footer presents the list of tools enabled on your site. (only for web version)
How to use it?
1- Add a tool to your Ido and select Menu and footer;
2- Add your company logo, with the option of jpg or png formats;

3 - Click + add item to insert the first Menu item. Available Options
  • Tool;
  • Store category (will only be enabled if the store function is active);
  • External link.

    • Add as many as you need.
4 - Define if you want to unpin the Menu from the top of the site, and if you want to activate the footer with the added items (available for Web version only).
When finished, click save.
