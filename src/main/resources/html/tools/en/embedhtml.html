What is it for?
HTML is a markup language used in website development. With this tool it is possible to write HTML codes in order to add functions to your Idolink with greater freedom.
How to use it?
1- Add a tool to your Ido and select ”HTML”;

2- Fill in the information on the page:
  • Title: Enter a title for the tool. This title is not displayed to the customer and will be used for internal control only;
  • HTML code: Paste the desired HTML code and save.
