¿Para qué sirve?
Esta herramienta le permite crear un formulario de suscripción para enviar boletines de Ido.
Esta característica es una excelente captura de clientes potenciales (datos de usuario) ya que completa y envía su información.
¿Cómo usarlo?
1- Agregue una herramienta a su Ido y seleccione "Formulario de boletín";

2- Rellena los datos de la pantalla:
  • Título: Ingrese un título para su formulario. Ejemplo: “Suscríbete a nuestra newsletter”, “Recibe novedades”;
  • Descripción: Explique brevemente lo que el usuario debe hacer en el formulario. Este campo es opcional;

3- Selecciona los campos opcionales que aparecerán para que el usuario complete:
  • Nombre;
  • Teléfono
El selector de correo electrónico ya estará activo, ya que es obligatorio para esta herramienta.
