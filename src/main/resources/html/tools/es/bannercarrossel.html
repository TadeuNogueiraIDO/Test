¿Para qué sirve?
Esta herramienta permite agregar múltiples imágenes, formando una galería de productos, servicios, fotografías, etc. .
También puede configurar un clic para cada imagen y al acceder, el usuario será dirigido a WhatsApp, E-mail o enlace externo.
Todavía es posible crear una página de detalles, con más información: título, descripción, precio y un botón de orientación.
¡Pero estad atentos! Con esta función no es posible realizar compras dentro de la aplicación. Para tener esta funcionalidad y toda la comodidad de ser una tienda Ido, adquiera la herramienta Tienda.
¿Cómo usarlo?
1- Agregue una herramienta a su Ido, seleccione "Carrusel de imágenes";

2- Ingrese un título para su carrusel y elija si será visible para el usuario;

3- Selecciona el formato de la imagen:
  • Retrato: imágenes rectangulares verticales;
  • Paisaje: imágenes rectangulares horizontales;
  • Cuadrado: mismo tamaño vertical y horizontalmente (1x1);
Es importante que el grupo de imágenes tenga el mismo formato para que no haya distorsión al aplicarlas al carrusel.
4- Sube las imágenes, seleccionando una a la vez desde tu dispositivo.

5- Definir si cada imagen tendrá su página de detalles. En caso afirmativo, complete:
  • Título;
  • Precio;
  • Descripción;
  • En el botón Página, seleccione una de las opciones:
    • Sin botón: solo página informativa;
    • Abra Whatsapp, luego complete el número de teléfono;
    • Abra el correo electrónico, luego ingrese el correo electrónico;
    • Ir a enlace externo: Introduzca la URL del sitio web o red social

En caso de que no desee crear las páginas de detalles, puede configurar la acción de clic para cada imagen:
  • Sin acción: sin clic;
  • Abra Whatsapp, luego complete el número de teléfono;
  • Abra el correo electrónico, luego ingrese el correo electrónico;
  • Ir a enlace externo: Ingrese la URL del sitio web o red social
