¿Para qué sirve?
HTML es un lenguaje de marcado utilizado en el desarrollo de sitios web. Con esta herramienta es posible escribir códigos HTML para añadir funciones a tu Idolink con mayor libertad.
¿Cómo usarlo?
1- Agrega una herramienta a tu Ido y selecciona ”HTML”;

2- Rellena los datos de la página:
  • Título: ingrese un título para la herramienta. Este título no se muestra al cliente y se utilizará solo para control interno;
  • Código HTML: pegue el código HTML deseado y guárdelo.
