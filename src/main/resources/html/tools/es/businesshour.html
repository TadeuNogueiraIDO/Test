¿Para qué sirve?
Esta herramienta te permite presentar el horario de apertura de tu establecimiento o incluso tu horario laboral. También puede crear una hora alternativa para una fecha o período específico, como feriados o vacaciones.
¿Cómo usarlo?
1- Agregue una herramienta a su Ido y seleccione "Horario de funcionamiento";

2- Introduce un título para el botón. Ej: Horario de apertura, acceda a nuestro horario ahora;

3-Selecciona si quieres el botón sin ícono, desde Ido Gallery (tenemos algunos íconos disponibles para ti en nuestra galería) o sube desde tu dispositivo (puede ser una foto, ícono o cualquier imagen).

4- Selecciona la hora:
  • Por definir: puede configurar manualmente los horarios, activando los días de la semana y agregando 2 o 4 horarios para cada día;
  • 24 horas: con esta opción habilitada, afirma que está trabajando todo el tiempo, todos los días;
5- Si es necesario, puede crear una hora alternativa para una fecha o período específico. Sobrescribirá temporalmente el definido anteriormente y caducará una vez finalizado.
También puede crear un mensaje de alerta para informar al usuario de esta hora alternativa. Aparecerá resaltado, justo encima de la otra información.
