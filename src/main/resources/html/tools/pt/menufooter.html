Para que serve?
O MENU ou cardápio é onde se concentra o que é de mais importante para apresentar ao usuário. É no MENU que podemos adicionar o acesso rápio para Ferramentas, Categorias de produtos e links externos.
O rodapé apresenta a lista de ferramentas habilitadas no seu site. (somente para versão Web)
Como usar?
1- Adicione uma ferramenta ao seu Ido e selecione Menu e rodapé;
2- Adicione o logotipo da sua empresa, com opção de formatos jpg ou png;

3 - Clique em + adicionar item para inserir o primeiro item do Menu. Opções disponíveis
  • Ferramenta;
  • Categoria da loja (somente será habilitada se a função loja estiver ativa);
  • Link externo.

    • Adicione quantos achar necessário.
4 - Defina se você quer desafixar o Menu do topo do site, e se quer ativar o rodapé com os itens adicionados (disponível para versão Web apenas).
Ao finalizar, clique em salvar.
