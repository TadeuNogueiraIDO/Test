Para que serve?
Esta ferramenta permite que você crie um formulário de inscrição para envio de newsletters Ido.
Esse recurso é um excelente captador de leads (dados de usuário), uma vez que ele preenche e envia suas informações.
Como usar?
1- Adicione uma ferramenta ao seu Ido e selecione "Formulário Newsletter”;

2- Preencha as informações da tela:
  • Título: informe um título para seu formulário. Exemplo: “Assine nossa newsletter”, "Receba novidades”;
  • Descrição: Explique resumidamente o que o usuário deve fazer no formulário. Esse campo é opcional;

3- Selecione os campos opcionais que aparecerão para o usuário preencher:
  • Nome;
  • Telefone
O seletor de E-mail já virá ativo, pois é obrigatório para essa ferramenta.
