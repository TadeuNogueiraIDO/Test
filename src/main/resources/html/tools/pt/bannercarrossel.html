Para que serve?
Essa ferramenta permite que você adicione várias imagens, formando uma galeria de produtos, serviços, fotografias etc.
Você também pode configurar um clique para cada imagem e ao acessar, o usuário será direcionado para WhatsApp, E-mail ou link externo.
Ainda é possível criar uma página de detalhes, com mais informações: título, descrição, preço e um botão de direcionamento.
Mas fique ligado(a)! Com esse recurso não é possível fazer compras dentro do app. Para ter essa funcionalidade e toda a comodidade de ser uma loja Ido, adquira a ferramenta Shop.
Como usar?
1- Adicione uma ferramenta ao seu Ido, selecione ”Carrossel de imagens”;

2- Informe um título para seu carrossel e escolha se ele ficará visível para o usuário;

3- Selecione o formato das imagens:
  • Retrato: imagens retangulares verticais;
  • Paisagem: imagens retangulares horizontais;
  • Quadrado: mesmo tamanho na vertical e horizontal (1x1);
É importante que o grupo de imagens tenha o mesmo formato para que não haja distorção quando aplicadas ao carrossel.
4- Faça upload das imagens, selecionando uma por vez do seu dispositivo.

5- Defina se cada imagem terá sua página de detalhes. Se sim, preencha:
  • Título;
  • Preço;
  • Descrição;
  • Em Botão da página selecione uma das opções:
    • Sem botão: página apenas informativa;
    • Abrir Whatsapp, então preencha o número de telefone;
    • Abrir e-mail, então informe o email;
    • Ir para link externo: Informe a URL do site ou rede social

Caso você não queira criar as páginas de detalhe, você pode definir a ação de clique para cada imagem:
  • Sem ação: não haverá clique;
  • Abrir Whatsapp, então preencha o número de telefone;
  • Abrir e-mail, então informe o email;
  • Ir para link externo: Informe a URL do site ou rede social
