Para que serve?
Esta ferramenta permite que você crie um formulário de ”Fale Conosco". Esse recurso é muito eficiente, pois cria um canal de comunicação entre o usuário do seu Ido e você.
No formulário, o usuário pode tirar dúvidas, fazer perguntas, enviar sugestões, além de ser um excelente captador de leads (dados de usuário), uma vez que ele precisa preencher os campos que você ativar.
Como usar?
1- Adicione uma ferramenta ao seu Ido e selecione "Fale Conosco”.

2-Preencha as informações da tela:
  • Título: informe um título para seu formulário. Exemplo: “Fale Conosco”, “Deixe sua dúvida comentário ou elogio” etc;
  • Descrição: Explique resumidamente o que o usuário deve fazer no formulário. Esse campo é opcional;
2-Selecione os campos opcionais que aparecerão para o usuário preencher:
  • Nome;
  • Telefone;
  • Assunto;
Os seletores de E-mail e Mensagem já virão ativos, pois são obrigatórios para essa ferramenta.
