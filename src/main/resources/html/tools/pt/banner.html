Para que serve?
Essa ferramenta permite que você adicione banners fixos de imagem ao seu Ido. Com esse recurso é possível exibir uma promoção, um anúncio ou a imagem de um produto ou serviço. Ainda é possível direcionar o usuário, através do clique, para o Whatsapp, e-mail, link externo ou apenas maximizar a imagem.
Como usar?
1- Adicione uma ferramenta ao seu Ido e selecione ”Banner Imagem”;

2- Informe um título para seu banner. Ele não aparecerá para o usuário, servirá apenas para organização;

3- Selecione uma das opções para quando o usuário clicar no banner:
  • Sem ação: não haverá clique no banner;
  • Abrir Whatsapp, então preencha informe o número de telefone no qual responderá no Whatsapp;
  • Abrir e-mail: depois informe o email referente;
  • Ir para link externo: Informe a URL do site ou rede social;
  • Maximizar imagem: o banner abrirá em tela cheia;

4- Clique em ”Upload” e selecione uma imagem do seu dispositivo.
