Para que serve?
HTML é uma linguagem de marcação utilizada no desenvolvimento de sites. Com essa ferramenta é possível gravar códigos HTML com a finalidade de acrescentar funções ao seu Idolink com maior liberdade.
Como usar?
1- Adicione uma ferramenta ao seu Ido e selecione ”HTML”;

2- Preencha com as informações da página:
  • Título: Informe um título para a ferramenta. Este título não é exibido para o cliente e será usado somente para controle interno;
  • Código HTML: Cole o código HTML desejado e salve.
